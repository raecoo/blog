比tail更实用的less

Aug 4th, 2009 | Comments 

tail -f development.log
less +F development.log

简单的看这两条命令没什么区别,但实际上后者比前者还是强大不少,起码需要在log里查找内容时不再需要C+c,然后用vim干活,而只需要简单的按一下C+c进入编辑模式,接下来的操作就和vim完全一样,hjkl, / 统统可以用上,完活后Shift+f返回log的监听模式.

出处: http://blog.libinpan.com/2009/07/less-is-better-than-tail/

# CentOS-Base.repo
#
# This file uses a new mirrorlist system developed by Lance Davis for CentOS.
# The mirror system uses the connecting IP address of the client and the
# update status of each mirror to pick mirrors that are updated to and
# geographically close to the client.  You should use this for CentOS updates
# unless you are manually picking other mirrors.
#
# If the mirrorlist= does not work for you, as a fall back you can try the 
# remarked out baseurl= line instead.
#
#

[base]
name=CentOS-$releasever - Base
mirrorlist=http://mirrorlist.centos.org/?release=$releasever&arch=$basearch&repo=os
#baseurl=http://mirror.centos.org/centos/$releasever/os/$basearch/
gpgcheck=1
gpgkey=http://mirror.centos.org/centos/RPM-GPG-KEY-CentOS-5

#released updates 
[updates]
name=CentOS-$releasever - Updates

#packages used/produced in the build but not released

mirrorlist=http://mirrorlist.centos.org/?release=$releasever&arch=$basearch&repo=addons
#baseurl=http://mirror.centos.org/centos/$releasever/addons/$basearch/

gpgkey=http://mirror.centos.org/centos/RPM-GPG-KEY-CentOS-5

在Leopard 卸载gem安装包

Jul 29th, 2009 | Comments

ERROR:  While executing gem ... (Gem::InstallError)
    cannot uninstall, check `gem list -d package_name`

google后发现需要指定一下安装路径就可以正常卸载了.

sudo gem uninstall net-ssh -v=1.1.2 --install-dir=/System/Library/Frameworks/Ruby.framework/Versions/1.8/usr/lib/ruby/gems/1.8/

解决Paperclip找不到Identify的问题

Jul 28th, 2009 | Comments

使用Paperclip生成缩略图时偶尔会发生无法找到identify的错误,原来解决过一次,是因不ImageMagick版本的问题,今天又碰以了这个问题,因为测试环境装了四个不同版本的ImageMagick,搞的有点头大,都无法正常使用.后来发现一个小开关,立马解决掉了,比较爽!为什么原来我在使用这个开关没有这样的效果,有点怪异.

开关就是在environment配置文件里添加如下设置:

Paperclip.options[:command_path] = "/path/to/imagemagick/bin/directory"

这个问题貌似在Passenger下比较明显,这与Passenger加载路径的方式也有一定的关系.
